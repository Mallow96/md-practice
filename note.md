# 筆記

## 教會職位

1. 職稱地位：大祀長 > 祀長 > 司祀 > 司次 > 候司
2. 候司等於是實習生一類的地位，負責協助司次以上的神職人員
3. 除了職稱差別，教會本身也有依規模而分。例如中央教會是主導一切教務的中心，因此中央教會大祀長是教會的總領導人
4. 因`小鎮`規模較小，沒有`大祀長`；最高職位僅到`祀長`。

## 角色

1. `他`：本地出身，是隸屬於`小鎮`本地`教會`的司祀
2. `她`：非本地出身，三年前因中央教會分發而來到`小鎮`，目前還是`候司`。
3. 孩子們：教會的主要職責之一便是照顧無家可歸的孩子；另外，不少貧窮家庭也會為了讓孩子得到更好的生活，而把孩子送到教會寄養。
4. `女孩`：無家可歸的`女孩`，從小就在`教會`生活，因此儘管年紀小，卻已經在這裡待了很久。從三年前，`她`來到教會起，兩人就處得特別好，`女孩`也把`她`當作姐姐看待。
5. `少女`：目前資歷最淺的女孩，才剛到教會生活幾周。

## 艾科爾果

1. 外型寬扁的圓球體水果，皮薄且光滑
2. 艾科爾果實變化：
   1. 冬末春初時開始結果，初期的顏色接近深藍
   2. 入春轉暖後，果實的顏色轉深紫
   3. 到了夏季，果實顏色繼續變淡、轉紅，果農便會在顏色呈現粉紅色時用刀具把果實從枝幹上挖下來採收
   4. 採收後，若放置到乾燥，果皮則會硬化並變成棕黑色，屆時吃起來口感與原本差距甚大，且風味濃郁。因此，人們常在天氣開始轉涼時，把艾科爾做成果乾保存，以便在冬季食用。
3. 艾科爾會在生長在果樹的枝幹上結果實，而不像一般水果那樣，會結於細枝末端

## 地窖

每年地窖會開啟兩次，都會由一群以農業公會為首的鎮民，和`教會`司祀以上的人員一起：

1. 深冬：鎮民會帶著一箱箱的貨物來到`教會`
2. 春天：隔幾個月後，同一批鎮民會再來搬走部分的貨品
